Задание:
Разработать программу, позволяющую шифровать и расшифровывать текст с использованием шифра Цезаря.

Режимы работы:

- Шифрование текста с помощью ключа
- Расшифровка текста с помощью ключа
- Расшифровка методом brute force
- Расшифровка методом статистического анализа

Дополнительные задачи:

 Валидация входных данных:

 - Ключ (введенное значение обязано быть числом. Число должно входить в определенный промежуток целых чисел)
 - Путь к файлу куда сохранить зашифрованное сообщение (путь должен быть валидным)
 - Путь откуда брать файл для расшифровки (файл по заданному пути должен существовать)

Уточнения:
 - Шифруются только определенные символы, остальные остаются неизменными.

   Список символов, которые шифруются (в нижнем и верхнем регистре) :
   'а', 'б', 'в', 'г', 'д', 'е', 'ж', 'з',
   'и','к', 'л', 'м', 'н', 'о', 'п', 'р', 'с', 'т', 'у', 'ф', 'х', 'ц', 'ч', 'ш', 'щ',
   'ъ', 'ы', 'ь', 'э', 'я', '.', ',', '«', '»', '"', '\'', ':', '!', '?', ' '
